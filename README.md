# Quadrado-Magico-Game

Jogo educativo — **Quadrado Mágico 3×3**. Puzzle matemático com níveis, drag-and-drop e progresso.

## Executar

```bash
cd "/Users/Mac/Quebra-cabeça 3x3"
python3 -m http.server 8080
```

Abrir http://localhost:8080

## Como jogar

1. Menu → **JOGAR** ou **SELECIONAR NÍVEL**
2. Arraste números (1–9) para a grelha ou toque para colocar
3. Complete quando todas as linhas, colunas e diagonais somarem **15**

Solução exemplo:
```
8  1  6
3  5  7
4  9  2
```
